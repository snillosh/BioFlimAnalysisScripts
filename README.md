# QR Generator

Self contained .NET console app for generating QR codes.

## Installation

Go to the releases section of this page and download the zip file that suites your operating systems.
For windows this is 'win-x64'. Once downloaded, unzip the contents to a sensible location. You should have an exe files that you can run to generate your QR code.

## Use

To use the QR code generator, follow the instructions in the command line interface.
